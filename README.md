# GameOfTile
move the empty tile to bring numbers in order

## How to run in your local?
 -> DOWNLOAD and UNZIP the file.  
 -> Install python  
 -> Install pygame package using ``pip install pygame``  
 -> Run the tileGame.py  
 -> Use arrow keys to move
